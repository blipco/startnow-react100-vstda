# VSTDA

## Description:

-- This application allows the user to create, edit and complete Todos

### Instructions:
###### Create New ToDo
1. Enter the text of your Todo into the "I want to.." text box.
2. Click the "Select A Priority" drop down box and select a priority level.
3. Click the ADD button to save your Todo.
###### Edit Todo
1. Click the Edit button on one of your saved Todos.
2. Make changes to your text and priority.
3. Click Save button to update the Todo.
###### Delete Todo
1. Click the delete button on one of your saved Todos.
###### Completed Todo
1. Once you have completed the Todo click the check box on the completed Todo.  This will strike a line through your Todo text.
